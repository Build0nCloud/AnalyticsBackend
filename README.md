# AnalyticsBackend
-------------------
req02.zip is the zip file that containes the python requests library.  You can use this file to create a Lambda layer for your functions that will allow you to import the requests and the requests_aws4auth librarys

The *.py files are the raw source for the lambda files.

The *.yaml files are exported SAM files that can be used in a CloudFormation Template.

The other *zip files are exported deployment packages that you can import directly into your account.  It should contain all the source and libraries combined.
